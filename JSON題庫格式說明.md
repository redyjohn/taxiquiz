# JSON 題庫格式說明

## 檔案結構

題庫 JSON 檔案應存放在 `public/data/` 目錄下，按照以下結構組織：

```
public/data/
├── 地理/
│   ├── 選擇題/
│   │   ├── 台中.json
│   │   ├── 彰化.json
│   │   └── 南投.json
│   └── 是非題/
│       ├── 台中.json
│       ├── 彰化.json
│       └── 南投.json
└── 法規/
    ├── 選擇題.json
    └── 是非題.json
```

## JSON 格式

### 選擇題格式

```json
{
  "category": "地理",
  "type": "選擇題",
  "region": "台中",
  "questions": [
    {
      "id": 1,
      "question": "題目內容",
      "options": ["選項A", "選項B", "選項C", "選項D"],
      "correctAnswer": 0
    },
    {
      "id": 2,
      "question": "另一題題目",
      "options": ["選項A", "選項B", "選項C", "選項D"],
      "correctAnswer": 2
    }
  ]
}
```

**說明：**
- `category`: 題目類別（"地理" 或 "法規"）
- `type`: 題型（"選擇題" 或 "是非題"）
- `region`: 地區名稱（僅地理類別需要，"法規"類別可為空字串）
- `questions`: 題目陣列
  - `id`: 題號（必須是數字，且必須連續或符合設定的範圍）
  - `question`: 題目內容
  - `options`: 選項陣列（選擇題通常為 4 個選項）
  - `correctAnswer`: 正確答案的索引（0 表示第一個選項，1 表示第二個選項，以此類推）

### 是非題格式

```json
{
  "category": "地理",
  "type": "是非題",
  "region": "台中",
  "questions": [
    {
      "id": 1,
      "question": "題目內容",
      "options": ["是", "否"],
      "correctAnswer": 0
    }
  ]
}
```

**說明：**
- 是非題的 `options` 固定為 `["是", "否"]`
- `correctAnswer`: 0 表示"是"，1 表示"否"

## 範例

### 地理-選擇題-台中

檔案路徑：`public/data/地理/選擇題/台中.json`

```json
{
  "category": "地理",
  "type": "選擇題",
  "region": "台中",
  "questions": [
    {
      "id": 1,
      "question": "台中市位於台灣的哪個位置？",
      "options": ["北部", "中部", "南部", "東部"],
      "correctAnswer": 1
    },
    {
      "id": 2,
      "question": "台中市是台灣的第幾大都市？",
      "options": ["第一大", "第二大", "第三大", "第四大"],
      "correctAnswer": 1
    }
  ]
}
```

### 法規-選擇題

檔案路徑：`public/data/法規/選擇題.json`

```json
{
  "category": "法規",
  "type": "選擇題",
  "region": "",
  "questions": [
    {
      "id": 1,
      "question": "計程車駕駛人執業登記證有效期限為幾年？",
      "options": ["3年", "4年", "5年", "6年"],
      "correctAnswer": 0
    }
  ]
}
```

## 題目範圍對應

- **彰化**: 題號 1-100

- **彰化**: 題號 1-100

### 法規-選擇題
- 題號 1-293

### 法規-是非題

## 注意事項

1. **題號必須連續**：每個 JSON 檔案中的題號應該從指定的起始號碼開始，並連續編號
2. **檔案編碼**：JSON 檔案請使用 UTF-8 編碼
3. **JSON 格式**：請確保 JSON 格式正確，可以使用線上 JSON 驗證工具檢查
4. **選項數量**：選擇題建議為 4 個選項，是非題必須為 2 個選項（"是"和"否"）
