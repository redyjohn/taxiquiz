# 計程車執業登記證題庫練習系統

這是一個使用 React + Vite 建立的計程車執業登記證題庫練習網站，支援 JSON 格式的題庫檔案。

## 功能特色

- 📚 支援多分類題庫（地理、法規）
- 📝 支援選擇題和是非題兩種題型
- 🗺️ 地理類別支援地區分類（台中、彰化、南投）
- 📊 即時答題反饋和結果統計
- 🎯 清楚的答題進度顯示
- 💾 使用 JSON 檔案管理題庫，易於擴充

## 技術棧

- **React 18** - 使用者介面函式庫
- **Vite** - 快速的前端建置工具
- **CSS3** - 現代化響應式設計

## 開始使用

### 安裝依賴套件

```bash
npm install
```

### 啟動開發伺服器

```bash
npm run dev
```

開發伺服器會啟動在 `http://localhost:5173`

### 建置生產版本

```bash
npm run build
```

### 預覽生產版本

```bash
npm run preview
```

## 題庫結構

系統支援以下題目分類：

### 一、地理

#### (一) 選擇題
- **台中**: 1-50題, 51-100題, 101-150題, 151-201題
- **彰化**: 1-50題, 51-100題
- **南投**: 1-50題, 51-102題

#### (二) 是非題
- **台中**: 1-50題, 51-100題, 101-150題, 151-197題
- **彰化**: 1-50題, 51-100題
- **南投**: 1-56題

### 二、法規

#### (一) 選擇題
- 1-50題, 51-100題, 101-150題, 151-200題, 201-250題, 251-293題

#### (二) 是非題
- 1-50題, 51-100題, 101-150題, 151-200題, 201-250題, 251-284題

## 題庫檔案格式

題庫 JSON 檔案應存放在 `public/data/` 目錄下，詳細格式請參考 [JSON題庫格式說明.md](./JSON題庫格式說明.md)

### 基本格式

```json
{
  "category": "地理",
  "type": "選擇題",
  "region": "台中",
  "questions": [
    {
      "id": 1,
      "question": "題目內容",
      "options": ["選項A", "選項B", "選項C", "選項D"],
      "correctAnswer": 0
    }
  ]
}
```

## 使用說明

1. **選擇題目範圍**：在首頁依序選擇類別、題型、地區（如有）和題目範圍
2. **開始練習**：點擊「開始練習」按鈕進入答題頁面
3. **答題**：選擇答案後會立即顯示正確答案和反饋
4. **瀏覽題目**：可使用「上一題」和「下一題」按鈕切換
5. **查看結果**：完成所有題目後可查看詳細的答題結果和統計

## 開發說明

- 編輯 `src/components/` 中的元件來修改介面
- 編輯 `src/data/questionStructure.js` 來修改題目結構
- 在 `public/data/` 目錄下放置 JSON 題庫檔案
