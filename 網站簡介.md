## 第一頁：專案概述

### 計程車執業登記證題庫練習系統

**專案簡介**

本系統是一個專為計程車執業登記證考試設計的線上題庫練習平台，提供完整的題目練習、模擬測驗與成績分析功能。系統採用現代化的前端技術架構，提供流暢的使用者體驗，幫助考生有效準備執業登記證考試。

**核心目標**

- 📚 提供完整的題庫資源（地理、法規兩大類別）
- 🎯 支援多種練習模式（選擇題、是非題、模擬測驗）
- 📊 即時答題反饋與詳細成績分析
- 💻 響應式設計，支援各種裝置使用
- 🚀 快速載入與流暢的使用體驗

**適用對象**

- 準備計程車執業登記證考試的考生
- 需要複習地理與法規知識的學習者
- 希望透過線上練習提升考試成績的使用者

## 第三頁：網站功能

### 完整功能介紹

**1. 歡迎頁面（Welcome Page）**

- 清晰的導航選單
- 三大主要功能入口：
  - 📖 題庫導讀
  - ✏️ 題庫練習
  - 📋 考試注意事項

**2. 題庫選擇系統（Home Page）**

- **多層級選擇介面**
  - 第一步：選擇類別（地理 / 法規）
  - 第二步：選擇題型（選擇題 / 是非題 / 模擬測驗）
  - 第三步：選擇地區（台中 / 彰化 / 南投，僅地理類別）
  - 第四步：選擇題目範圍（如：1-50題、51-100題等）

- **智能範圍選擇**
  - 根據選擇的類別、題型、地區動態顯示可用範圍
  - 模擬測驗模式：自動隨機抽題（選擇題25題 + 是非題25題）

**3. 答題頁面（Quiz Page）**

- **即時答題體驗**
  - 清晰的題目顯示
  - 選項按鈕互動效果
  - 答題進度條顯示
  - 當前題目編號提示

- **導航功能**
  - 上一題 / 下一題按鈕
  - 自動跳轉到下一題（選擇答案後）
  - 完成測驗按鈕

- **答題狀態管理**
  - 即時保存答題記錄
  - 支援隨時修改答案
  - 顯示已答題目狀態

**4. 結果分析頁面（Quiz Result）**

- **成績統計**
  - 總分顯示（百分比）
  - 答對題數 / 總題數
  - 成績等級評定（優秀 / 良好 / 及格 / 需要再加強）
  - 視覺化分數圓圈顯示

- **錯題詳情**
  - 列出所有錯誤題目
  - 顯示您的答案 vs 正確答案
  - 題目編號與內容完整顯示

- **後續操作**
  - 錯題再回答功能
  - 重新測驗功能
  - 返回首頁功能

**5. 題庫導讀（Guide Page）**

- 詳細的題庫結構說明
- 使用方式教學
- 題目分類介紹

**6. 考試注意事項（Notice Page）**

- 考試相關重要資訊
- 注意事項提醒
- 準備建議

**7. 模擬測驗功能**

- **地理模擬測驗**
  - 從台中、彰化、南投三個地區的題庫中隨機抽取
  - 選擇題 25 題 + 是非題 25 題（共 50 題）
  - 題目順序隨機打亂

- **法規模擬測驗**
  - 從法規題庫中隨機抽取
  - 選擇題 25 題 + 是非題 25 題（共 50 題）
  - 題目順序隨機打亂
