## 使用方法

### 方式一：使用 AdBanner 組件（推薦）

在需要顯示廣告的位置，傳入 `adSlot` 屬性：

```jsx
// 橫向廣告 (728x90)
<AdBanner position="horizontal" adSlot="你的廣告位置ID" />

// 豎向廣告 (300x250)
<AdBanner position="vertical" adSlot="你的廣告位置ID" />

// 正方形廣告 (300x300)
<AdBanner position="square" adSlot="你的廣告位置ID" />
```

### 方式二：直接在組件中使用（進階）

如果需要自訂廣告，可以在 `AdBanner.jsx` 的 `ad-content` div 中直接添加：

```jsx
<div className="ad-content">
  <ins
    className="adsbygoogle"
    style={{display:'inline-block',width:'728px',height:'90px'}}
    data-ad-client="ca-pub-4218582490253078"
    data-ad-slot="你的廣告位置ID"
  />
</div>
```

## 設定步驟

1. 登入 [Google AdSense](https://www.google.com/adsense/)
2. 進入「廣告」→「按廣告單元」頁面
3. 創建新的廣告單元
4. 選擇廣告格式：
   - 橫向：728x90 (領袖版位)
   - 豎向：300x250 (中矩形)
   - 正方形：300x300
5. 複製「廣告單元 ID」
6. 在組件中傳入 `adSlot="廣告單元ID"`

## 注意事項

- 廣告需要 Google 審核通過後才會顯示
- 審核期間會顯示占位符
- 每個廣告位置可以使用不同的 `adSlot`
- 確保遵守 Google AdSense 政策

## 範例

```jsx
// 首頁頁首廣告
<AdBanner 
  position="horizontal" 
  className="ad-header" 
  adSlot="1234567890" 
/>

// 結果頁面側邊欄廣告
<AdBanner 
  position="vertical" 
  className="ad-sidebar" 
  adSlot="0987654321" 
/>
```
